# Kotlin Memo App

Androidの基礎を学習するためのシンプルなメモアプリケーション

## 概要
Android開発の基本要素を習得するための学習プロジェクトです。
Activity、RecyclerView、SharedPreferencesなどの基本的な概念を実践的に学びます。

## 機能
- [ ] メモの一覧表示
- [ ] メモの追加
- [ ] メモの編集
- [ ] メモの削除
- [ ] データの永続化（SharedPreferences）
- [ ] 検索機能

## 技術スタック
- Kotlin
- Android SDK (API 24+)
- RecyclerView
- SharedPreferences
- ViewBinding
- Material Design

## 学習目標
- ActivityとFragmentのライフサイクル
- XMLレイアウト基礎
- ViewとViewGroup
- Intent、Bundle
- SharedPreferencesでのデータ永続化
- リソース管理（strings.xml、colors.xml等）

## 開発環境
- Android Studio
- Minimum SDK: API 24 (Android 7.0)
- Target SDK: API 34

- [ ] Week 2: 詳細画面、CRUD操作、画面遷移
